# -QT-Player-developed-based-on-Qt
基于QT开发的播放器，开发框架采用FFmpeg3.6和SDL2.0  
主要功能：  
1、播放本地音视频文件  
2、播放网络串流（RTSP,RTMP...)  
次要功能：  
1、文件播放、暂停、停止  
2、进度跳转  
3、音量设置  
4、倍速设置
